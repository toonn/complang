Exercise session Functional Reactive Programming (Elm)

In this session, you will learn to work with Functional Reactive Programming (FRP) and the Elm programming language.

The session consists of 6 exercises, which you can find in the same zip-file as this assignment, under the names Oef1.elm to Oef6.elm. Each file contains the assignment for the exercise.

Working with Elm

To use Elm, unzip the assignments to a directory cpl-elm-oefenzitting in your home directory. Then open a Terminal and run the following command:

cd ~/cpl-elm-oefenzitting/
elm-reactor

Then minimize the Terminal screen, but DO NOT CLOSE IT. You can then open a browser at this URL and either run or debug the programs in that directory. We recommend you edit the elm files using editor Kate or GEdit and set highlighting mode to Haskell. If you debug an application, it is by default hotswapped, i.e. your changes are directly reflected in the running application when you save.
